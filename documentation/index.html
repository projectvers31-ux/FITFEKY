Header Customization

Logo Change:

Header logo of the site should be changed in Header section of the target page with the header tag.

Adding a new menu item:

To add a new menu item to an existing menu, you should define a new list element on the first navigation level.

Making menu item active:

To make a menu item active, use .active class for a target element.

Adding a submenu:

To add a submenu to the existing menu item, create an additional list with .sub-menu class inside the corresponding menu item.

Classes Area

You can add/remove item from the Classes section.

Changing Image/Link

Changing image:

The src attribute contains a path pointing to the image you want to embed in the page. To change the images that are in html just add the image in the images folder and then change the value of img src in HTML.

CSS - Modification:

To change the images that are in css just add the image in the images folder and then change the name of background-image in CSS.

Contact Form

Creating a working contact form in HTML using PHP involves two main steps:

  1. Creating the HTML form
  2. Writing the PHP script to process form data
Step 1: Create the HTML Form (contact.html)
	
<form class="contact-form" method="post" action="contact-handler.php">
	<input type="text" name="username" placeholder="Your Name">
	<input type="email" name="usermail" placeholder="Your Email">
	<input type="tel" name="userphone" placeholder="Phone Number">
	<textarea id="usermessage" name="usermessage" placeholder="Enter your messages"></textarea>
	<button type="submit">Submit Questions<i class="fa-solid fa-arrow-right"></i></button>
</form>
Step 2: Create the PHP Processing Script (contact-handler.php)
	
<?php
	$name = $_POST['username'];
	$visitor_email = $_POST['usermail'];
	$visitor_phone = $_POST['userphone'];
	$message = $_POST['usermessage'];
	
	$email_from='Fitcore';
	$email_subject="New Contact";
	$email_body="User Name:$name.\n".
				"User Email:$visitor_email.\n".
				"User Phone:$visitor_phone.\n".
				"User Message:$message.\n";
	
	$to="info@example.com";
	$headers="From:$email_from\r\n";
	$headers.="Reply-To:$visitor_email\r\n";

	mail($to, $email_subject, $email_body, $headers) or die("Error!");

	header("Location:thank-you.html");

	exit;
?>
Debugging Tips:

  • Ensure PHP mail is configured on your server. If using a local server (XAMPP, WAMP), you may need an SMTP setup (e.g., using PHPMailer).
  • Always validate and sanitize user input before processing.
  • Use error_reporting(E_ALL); ini_set('display_errors', 1); in your PHP file for debugging errors.
